# Code of Conduct

## Our Promise

We want VeilClip to be an open-source project where people can learn, build, and help in a safe and respectful way.

Everyone who joins this project should feel welcome, including people of every:

- age
- background
- identity
- experience level
- language
- nationality
- religion

## Expected Behavior

Please:

- be respectful
- be patient
- be clear
- assume good intent first
- give helpful feedback
- accept feedback with respect
- focus on what is best for the project

## Unacceptable Behavior

Please do not:

- insult or attack people
- harass anyone
- use hateful language
- post sexual content or threats
- share private information without permission
- troll, bait, or intentionally derail discussions
- spam issues, pull requests, or discussions

## Scope

This code of conduct applies to:

- issues
- pull requests
- code review
- project discussions
- community spaces linked to this project

## Enforcement

Project maintainers may:

- ask for a behavior change
- edit or remove harmful content
- close discussions
- reject contributions
- temporarily or permanently ban people from project spaces

They will try to act fairly and protect the community.

## Reporting

If someone breaks this code of conduct, report it privately:

- Email: `osenpacom@gmail.com`

Please include:

- what happened
- where it happened
- links or screenshots if possible

Reports will be reviewed carefully and handled as privately as possible.

## Project Standard

By joining this project, you agree to follow this code of conduct.
